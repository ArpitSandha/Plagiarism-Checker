# Plagiarism Checker using TF-IDF

## 📌 Overview
This project detects similarity between text files using TF-IDF vectorization and cosine similarity.

## 🚀 Features
- Compare multiple text files
- Detect similarity percentage
- Uses ML-based approach

## 🛠 Tech Stack
- Python
- scikit-learn (TF-IDF, cosine similarity)
